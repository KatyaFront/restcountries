# Rest countries

## Установка

1. Клонируйте репозиторий:
   `git clone ____`

2. Установите зависимости:
   `npm install`

## Использование

1. Запустить проект в режиме разработки:
   `npm run dev`

2. Собрать проект для продакшена:
   `npm run build`

3. Запустить локальный сервер для предварительного просмотра собранного проекта:
   `npm run preview`
